# Comic
##一款漫画阅读的app软件

1、网络框架使用的是Retrofit2
  灵活设置网络缓存时间：支持单独设置接口缓存过期时间，未设置缓存过期时间的走默认值过期时间。
  
2、图片加载框架使用的是facebook家的fresco
  对fresco进行简单封装；支持可自定义图片进度加载，自定义本地缓存位置、大小。

3、自定义ScaleRecyclerView，支持RecyclerView的缩放（双击、手势），拖动。

###手势缩放有待优化，希望大神可以指点。
